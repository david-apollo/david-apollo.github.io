如何搭建HEXO博客 mac版

Posted on 2020-09-11 Edited on 2020-09-12

1. 安装node

  • 下载：https://nodejs.org/zh-cn/

2. 安装git

  • 下载：https://git-scm.com/downloads

3. 安装hexo

  • 官方文档：https://hexo.io/zh-cn/docs/#%E5%AE%89%E8%A3%85-Hexo
  • npm install -g hexo-cli
  • hexo的两种执行方式
    • 第一种：npx hexo
    • 第二种：
      • echo ‘PATH=”$PATH:./node_modules/.bin”‘ >> ~/.bash_profile
      • source ~/.bash_profile
      • hexo
  • 可能遇到的问题：
    • hexo: command not found
      • 原因：出现这种情况主要是node的版本问题
      • 解决方案：通过nvm来控制一下node的版本来解决

4. 安装nvm

  • curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
  • source ~/.bash_profile

5. 关联github仓库

  • 创建一个名为.github.io的仓库，username指的是你github的用户名
  • 在本地电脑合适的位置创建一个.github.io文件夹
  • 在文件夹中执行以下命令
    • hexo init
    • npm install
    • hexo s (运行程序，访问本地localhost:4000可以看到博客已经搭建成功)
  • 修改文件夹中的_config.yml文件, repository为github项目仓库地址

6. 如何多台电脑同时维护博客

  • 在github仓库新建一个hexo分支用来存储博客代码，在.github.io文件夹中执行以下命令
    • git init
    • git add .
    • git commit -m “xxx”
    • git remote add origin https://github.com/XXX/XXX.github.io.git
    • git branch hexo
    • git push -u origin hexo
    • git branch –set-upstream-to=origin/hexo master
  • 以后每台电脑都可以通过hexo分支进行协作

7. 一些hexo博客自定义相关操作

  • 更换主题
    • 通过git clone 将相关主题拉取到theme文件夹下
    • 修改_config.yml中的theme为你刚刚拉取的theme对应名称
    • 重新生成博客静态文件，发布到github即可，访问博客主题变化会有一定的延迟
